# ARTIFICA Portfolio - Лендинг с админ-панелью

FastAPI приложение для отображения портфолио проектов с админ-панелью для управления записями.

## Возможности

- Публичный лендинг с отображением проектов
- Админ-панель для управления проектами (CRUD операции)
- Загрузка изображений для проектов
- Аутентификация админа через пароль
- Локальная база данных SQLite

## Установка

1. Убедитесь, что у вас установлен Python 3.12 или выше

2. Установите зависимости:
```bash
pip install -e .
```

Или используйте uv:
```bash
uv pip install -e .
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Отредактируйте `.env` и установите пароль админа:
```
ADMIN_PASSWORD=your-secure-password-here
SECRET_KEY=your-secret-key-change-in-production
```

## Запуск

Запустите приложение с помощью uvicorn:

```bash
uvicorn app.main:app --reload
```

Приложение будет доступно по адресу: http://localhost:8000

## Использование

### Публичный доступ

- Главная страница: http://localhost:8000
- JSON API проектов: http://localhost:8000/api/projects

### Админ-панель

1. Перейдите на http://localhost:8000/admin/login
2. Введите пароль из файла `.env`
3. После входа вы попадете на дашборд: http://localhost:8000/admin/dashboard
4. Используйте кнопку "Добавить проект" для создания нового проекта
5. Редактируйте или удаляйте проекты через соответствующие кнопки

### Структура проекта

При создании проекта заполните следующие поля:

- **Название проекта** - заголовок проекта
- **Отрасль** - отрасль проекта (например, "Знакомства, соцсети, коммуникации")
- **Результаты** - список достижений (каждый результат с новой строки)
- **Сроки** - сроки проекта (например, "1-3 месяца")
- **Бюджет** - бюджет проекта (например, "18 000$")
- **Выгода для клиента** - описание выгоды
- **Технологический стек** - можно добавить несколько технологий (название и значение)
- **Изображения** - загрузите изображения проекта

## Структура базы данных

База данных SQLite создается автоматически при первом запуске в файле `projects.db`.

Таблица `projects` содержит все проекты с полями:
- id, title, industry, results, timeline, budget, benefits, tech_stack, images, created_at, updated_at

## Разработка

Проект использует:
- FastAPI для веб-фреймворка
- SQLAlchemy для работы с БД
- Jinja2 для шаблонов
- Pydantic для валидации

Подробная документация архитектуры находится в файле `architecture.md`.
